Script 2 Input Format

Script 2 uses an advanced CSV format with multiple columns:

  • keyword: The main keyword for the article (required)
  • featured_img: The keyword for the featured image (optional)
  • subtitle1, subtitle2, etc.: Custom section headings (optional)
  • img1, img2, etc.: Keywords for section images (optional)
